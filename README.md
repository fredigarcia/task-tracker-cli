# task-tracker-cli
Task Tracker CLI build a cli app to track task and manage to do list
Note, the json file will be created after adding the firsk task
Note, when the script in the example below include a number it refers to the id

run the script to check the help menu
python task.py

to add a task
python task.py add "Buy milk"

to list all tasks
python task.py list

to mark a task as in progress
python task.py mark-in-progress 1

to mark a task as done
python task.py mark-done 1

to update a task
python task.py update 2 "Finish FastAPI homework"

to delete a task
python task.py delete 1
